Using String in python

  1. A simple string is denoted by apostrophe ""
  2. Strings in python are immutable, therefore if you want to change or modify an existing string, you can only do so by adding a new string to the existing variable, and then return the value to a new variable by assigning a new variable.
  3. If you want to use special characters or single apostrophe within a string, you can either have double quotation marks outside the string or vice versa. And if you require both double and single quotation marks within a string, then use """ triple quotation marks to enclose the string.
  4. Multiline: you can use a \n to add the text in a new line, or simply enter to a new line without using \n, but only within triple quotation marks.
  5. There are various methods that work with strings such as title() that allows you to make your string sentence case, and split() that allows you to separate each word into a list
  6. Split() method allows you to split every string of a variable that has a space in between.
  7. Search you can search a word or character within a string without using a loop to check. example=> print("moon" in "the moon shines brightly today.")
  8. find() method The .find() method returns a -1 when the word isn't found, or it returns the index (the number representing the place in the string).
  9. count() methodallow python to ccount the number of times a word/string occurs.
  10. Strings are case sensitive, which means that if you are trying to find a word that has capital letters, the output will be false, so you need to first use a method to convert string to lower case by lower() or uppercase by upper() method., then define your search accordingly.
