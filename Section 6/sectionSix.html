Arithmetic operations

  1. Addition, subtraction, division, multiplication
  2. floor division: enables you to divide 2 numbers and provide an output in integer instead of decimals. It rounds down the decimal output.
  3. Remainder operator % provides you the output by first dividing the 2 numbers, then providing an output of the remainder value. Example 1000 % 60 => 40
  4. Order of operations PEMDAS =>
    1. Parenthesis
    2. Exponents
    3. multiplication
    4. Division
    5. Addition
    6. Subtraction
  5. Integers : converting a string number to an integer, or whole number use the keyword int
  6. Float is used when you want to convert a string number in decimal to a decimal number
  7. abs is used for an absolute value. Example you want to determine the distance between 2 planets, it cannot be in a negative number, so no matter what the numbers are, the output will remain absolute or a positive value, even if it contains a negative sign.
  8. Rounding if the value has a decimal of 0.5, then python rounds it up OR down to its nearest even integer.
  9. import MATHS library allows you to use functions like ceil, and floor and other libraries like pi.
  10. input you can use input to take the input of a user by input("Enter your number?"). Remember, input returns string values
