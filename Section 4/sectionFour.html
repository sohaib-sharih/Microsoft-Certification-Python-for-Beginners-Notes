Use boolean logic for Python

  1. Boolean is used to write conditional logic.
  2. Will help create conditional logic to take decision driven programs.
  3. Mathematical operators:
    1. equals ===
    2. less than or equal to < = < /li>
    3. not equal !=
  4. test expressions are written in if statements to check if the code is true or false, if its true, then the next code block will run otherwise it will not.
  5. in Python, NONE, 0 or false, all mean the same thing.
  6. The body of the if statement should be indented, if its not, then python will run that code even if the if statement was false.

Else, elif

  1. You can add multiple test expressions in an if statement incase if the test is false, then you would like th eprogram to do something else using Elif.
  2. Elifis a short form of else-if statement. This enables you to add multiple / additional test expressions.
  3. Once the condition is true in any particular stage of the if statement, then the next code block won't run. It keeps running to the next line until a condition is true. else allows you to make the program perform an action/output if all the conditions are false.
  4. Nested conditional logic: all rules of a regular if statement apply the same way.
  5. You can have an if condition indented inside an if statement.

And and OR operators

  1. If you want to combine multiple conditions inside a single test expression, then you can use the keywords Operators of AND and OR
